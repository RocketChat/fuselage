## Install

<!--install-->

Firstly, install the peer dependencies (prerequisites):

```sh
npm i react react-dom

# or, if you are using yarn:

yarn add react react-dom
```

Add `@rocket.chat/fuselage-ui` as a dependency:

```sh
npm i @rocket.chat/fuselage-ui

# or, if you are using yarn:

yarn add @rocket.chat/fuselage-ui
```
