## Install

<!--install(dev)-->

Firstly, install the peer dependencies (prerequisites):

```sh
npm i -D prettier

# or, if you are using yarn:

yarn add -D prettier
```

Add `@rocket.chat/prettier-config` as a dependency:

```sh
npm i -D @rocket.chat/prettier-config

# or, if you are using yarn:

yarn add -D @rocket.chat/prettier-config
```

<!--/install(dev)-->
