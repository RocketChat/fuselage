## Install

<!--install-->

Add `@rocket.chat/fuselage-tokens` as a dependency:

```sh
npm i @rocket.chat/fuselage-tokens

# or, if you are using yarn:

yarn add @rocket.chat/fuselage-tokens
```

<!--/install-->

